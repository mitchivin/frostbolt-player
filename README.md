## Overview
Frostbolt replaces the cluttered, duplicate-heavy interface of traditional IPTV applications with a crisp, desktop-optimised experience. It provides immediate channel rendering, intelligent background guide loading, and granular EPG customisation directly from your settings. All core user features and layouts are documented in plain English within the features-layman.md file.

## Key Features
* **Smart Channel Merging:** Combines duplicate SD, HD, and backup streams into a single clean row, letting you pick or cycle through backup feeds if one drops out.
* **Automated Stream Failover:** If a live feed stalls or disconnects, Frostbolt automatically switches to your next backup media source without interrupting your viewing.
* **Deep EPG Control:** Adjust guide data settings globally, per-provider, or on an individual channel basis using interactive time-offset sliders.
* **Foreground-First Loading:** The full-screen opening spinner dismisses the moment your first few visible channels are ready, letting you watch TV while the rest of the library finishes loading in the background.
* **Portable Library Blueprints:** Save or restore your custom channel merges, names, and EPG links as a lightweight file without ever exposing your private stream passwords.
